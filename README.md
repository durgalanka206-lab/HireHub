# HireHub

HireHub is a full-stack MERN job portal application that connects job seekers and recruiters.

## Features

- User Authentication
- Job Listings
- Apply for Jobs
- Admin Dashboard
- Responsive UI
- MongoDB Integration

## Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
