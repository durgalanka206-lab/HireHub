# HireHub Backend

Backend API for HireHub Job Portal built using the MERN stack.

## Features

- User Authentication
- JWT Authorization
- Job Management
- Apply for Jobs
- Admin Dashboard
- MongoDB Integration
- REST API Architecture

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## Installation

```bash
git clone <repo-url>
cd hirehub-backend
npm install
npm start
```

## Environment Variables

Create a .env file:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```
