# HireHub

HireHub is a job portal web application built using MERN stack.

## Features
- User Authentication
- Job Listings
- Apply for Jobs
- Admin Dashboard
- Responsive UI

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
