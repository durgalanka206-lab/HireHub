# HireHub Frontend

Frontend application for HireHub, a MERN stack job portal connecting job seekers and recruiters.

## Features

- User Authentication
- Browse Job Listings
- Search and Filter Jobs
- Apply for Jobs
- Responsive UI
- Backend API Integration

## Tech Stack

- React
- Vite
- JavaScript
- Axios
- CSS

## Installation

```bash
git clone https://github.com/durgalanka206-lab/HireHub.git
cd hirehub-frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000
```
